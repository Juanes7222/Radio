# Project Context

This repository is a TypeScript-first project.

## Confirmed stack

- Package manager: pnpm
- Backend: Express
- ORM: Prisma
- Frontend: React
- Styling: Tailwind CSS
- Tests: not defined yet

## Working assumptions

- Prefer TypeScript over JavaScript for new work.
- Keep backend logic in Express route handlers, services, and Prisma-access layers.
- Keep frontend components small and focused.
- Avoid inventing a test setup that the repository does not already use.
