Use this skill when working inside this repository.

## Core expectations

- Respond in Spanish.
- Do not use emojis.
- Prefer TypeScript over JavaScript.
- Write production-ready, maintainable code.
- Keep comments minimal and in English.
- Keep docstrings and JSDoc concise and in English.
- Avoid any unless strictly necessary.

## Stack

- pnpm
- Express
- Prisma
- React
- Tailwind CSS

## Change strategy

- Read the existing codebase first.
- Make incremental changes.
- Preserve project conventions.
- Avoid unrelated refactors.
- Call out risks and edge cases clearly.
- Suggest tests when logic changes.
