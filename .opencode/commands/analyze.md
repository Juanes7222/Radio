Read the current repository context carefully.

Use the project instructions and the context files first:
- `AGENTS.md`
- `docs/project-context.md`
- `docs/backend-context.md`
- `docs/frontend-context.md`

Identify:
- the relevant files
- the current architecture
- the types and modules involved
- the safest implementation path
- edge cases and risks
- whether tests should be added or updated

Do not modify code.
Return a concise but complete implementation plan.
